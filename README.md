# Business Performance KPI Forecasting & Executive Insights Dashboard

## Overview
End-to-end business analytics system using SQL, Python, and Power BI 
to forecast revenue and deliver executive-level KPI insights for a 
US retail company across 4 years of sales data.

## Tools
- SQL (SQLite)
- Python (Pandas, Prophet, Matplotlib)
- Power BI
